¿Qué es Amazon EC2? ¿Qué hace? ¿Qué problema resuelve? ¿Qué beneficios tiene? Estas son algunas de las preguntas que muchos se hacen al escuchar hablar de este servicio de computación en la nube de Amazon Web Services (AWS). En este artículo, vamos a intentar responder a estas cuestiones y dar una visión general de Amazon EC2, sus características, sus ventajas y sus casos de uso.

Amazon EC2 (Elastic Compute Cloud) es un servicio que proporciona capacidad informática en la nube segura y de tamaño modificable. Esto significa que permite lanzar servidores virtuales (llamados instancias) en la nube de AWS, con la posibilidad de escalar la capacidad según las necesidades de cada momento. Así, se puede acceder a infraestructura confiable y escalable bajo demanda, sin tener que invertir en hardware propio ni gestionar su mantenimiento.

Amazon EC2 ofrece la plataforma de computación más amplia y completa, con más de 700 instancias y la posibilidad de elegir el procesador, almacenamiento, redes, sistema operativo y modelo de compra más reciente, en función de lo que mejor se ajuste a las necesidades que plantea la carga de trabajo. Es el primer proveedor de servicios en la nube destacado que admite procesadores Intel, AMD y ARM, la única nube con instancias Mac de EC2 bajo demanda y la única nube con redes Ethernet de 400 Gbps. Además, ofrece el mejor rendimiento por precio para la formación con machine learning, así como el menor costo por instancias de inferencia en la nube.

Algunas de las características más destacadas de Amazon EC2 son:

- Imágenes de Amazon Machine (AMI): son plantillas preconfiguradas para las instancias que empaquetan los componentes que se necesitan para el servidor (incluido el sistema operativo y el software adicional).
- Tipos de instancias: son varias configuraciones de CPU, memoria, almacenamiento, capacidad de red y gráficos para las instancias. Se pueden elegir entre familias optimizadas para diferentes propósitos: computación general, computación optimizada, memoria optimizada, almacenamiento optimizado, aceleración optimizada o gráficos optimizados.
- Volumenes: son unidades de almacenamiento persistentes para los datos que se utilizan con las instancias. Se pueden elegir entre diferentes tipos: SSD (de propósito general o provisionado), HDD (de rendimiento magnético o frecuente) o EBS (Elastic Block Store), que ofrece un alto rendimiento y durabilidad.
- Grupos de seguridad: son firewalls virtuales que controlan el tráfico de entrada y salida de las instancias. Se pueden definir reglas para permitir o denegar el acceso a determinados puertos o protocolos.
- Elastic Load Balancing: es un servicio que distribuye el tráfico entrante entre varias instancias para mejorar la disponibilidad y el rendimiento.
- Auto Scaling: es un servicio que ajusta automáticamente el número de instancias en función del nivel de demanda o de las condiciones definidas por el usuario.
- Elastic IP: son direcciones IP públicas estáticas que se pueden asociar a las instancias. Permiten mantener una dirección IP fija aunque se cambie o reinicie la instancia.
- Amazon VPC (Virtual Private Cloud): es un servicio que permite crear una red virtual aislada en la nube de AWS. Dentro de una VPC se pueden lanzar las instancias y configurar aspectos como subredes, tablas de ruteo, gateways o VPN.

Amazon EC2 resuelve el problema de tener que comprar, instalar y administrar servidores físicos para ejecutar aplicaciones. Con Amazon EC2 se puede acceder a recursos informáticos bajo demanda, pagando solo por lo que se usa. Esto supone una serie de beneficios como:

- Reducir los costos: al no tener que invertir en hardware propio ni pagar por su mantenimiento, se puede ahorrar dinero y tiempo. Además, se puede aprovechar los modelos de compra flexibles que ofrece Amazon EC2, como las instancias reservadas, las instancias spot o los planes de ahorro.
- Aumentar la agilidad: al poder lanzar instancias en minutos y escalar la capacidad en cuestión de segundos, se puede responder rápidamente a los cambios en las necesidades del negocio o del mercado. Además, se puede acceder a la última tecnología y a las mejores prácticas de AWS.
- Mejorar la seguridad: al contar con la protección de AWS, que cumple con los más altos estándares de seguridad y privacidad. Además, se puede controlar el acceso a las instancias mediante grupos de seguridad, claves, roles y políticas.
- Optimizar el rendimiento: al poder elegir el tipo de instancia más adecuado para cada carga de trabajo y beneficiarse de las características de alto rendimiento que ofrece Amazon EC2, como las redes de 400 Gbps, las instancias basadas en AWS Graviton o las instancias Mac.
- Innovar más: al poder dedicar más tiempo y recursos al desarrollo y la mejora de las aplicaciones, en lugar de a la gestión de la infraestructura. Además, se puede integrar fácilmente con otros servicios de AWS, como Amazon S3, Amazon RDS o Amazon SageMaker.

Amazon EC2 se puede utilizar para ejecutar todo tipo de aplicaciones, desde las más simples hasta las más complejas y exigentes. Algunos de los casos de uso más comunes son:

- Ejecutar aplicaciones nativas en la nube y empresariales: Amazon EC2 ofrece infraestructura de computación segura, confiable, de alto rendimiento y rentable para cumplir necesidades empresariales exigentes. Se pueden migrar las aplicaciones empresariales existentes a AWS o crear nuevas aplicaciones nativas en la nube aprovechando los servicios y herramientas que ofrece AWS.
- Escalar para aplicaciones de computación de alto rendimiento (HPC): Amazon EC2 permite acceder a la infraestructura y capacidad bajo demanda que se necesita para ejecutar aplicaciones de computación de alto rendimiento (HPC) de forma más rápida y rentable. Se pueden utilizar instancias optimizadas para HPC, redes de baja latencia y alto ancho de banda, almacenamiento rápido y escalable, y servicios como AWS Batch o AWS ParallelCluster.
- Desarrollar para plataformas de Apple: Amazon EC2 permite crear, probar y firmar cargas de trabajo de macOS bajo demanda. Se puede acceder a entornos en minutos, escalar la capacidad dinámicamente según las necesidades y beneficiarse de los precios con pago por uso de AWS. Se pueden utilizar instancias EC2 Mac para desarrollar aplicaciones para iOS, iPadOS, macOS, watchOS y tvOS.
- Formar e implementar aplicaciones de machine learning: Amazon EC2 ofrece la mayor oferta de servicios de computación, redes (hasta 400 Gbps) y almacenamiento creados a medida para optimizar el rendimiento por precio para proyectos de machine learning. Se pueden utilizar instancias optimizadas para machine learning, como las basadas en AWS Graviton o las que incorporan GPU o inferentia.

Con Amazon EC2 se puede reducir los costos, aumentar la agilidad, mejorar la seguridad, optimizar el rendimiento e innovar más.
